# 🎬 CineBluish — OTT Streaming App

CineBluish is a modern OTT-style movie streaming mobile application built using **React Native + Expo + TypeScript** with real-time movie data powered by the **TMDB API**.

The app delivers a premium streaming experience inspired by platforms like Netflix and Prime Video with features such as dynamic movie browsing, cinematic hero banners, watchlists, trailers, continue watching, advanced search, and smooth UI interactions.
