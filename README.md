# 🎬 CineBluish — Premium OTT Streaming App

CineBluish is a modern OTT-style movie streaming mobile application built using **React Native + Expo + TypeScript** with real-time movie data powered by the **TMDB API**.

The app delivers a premium cinematic streaming experience inspired by platforms like Netflix, Prime Video, and Disney+ with features such as dynamic movie discovery, animated onboarding, cinematic splash screen, reactive watchlists, advanced search architecture, smooth transitions, and immersive UI interactions.
